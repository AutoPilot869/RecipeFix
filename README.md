# RecipeFix – Inline ingredient badges for recipe directions. Tap to mark done.

RecipeFix is a Chrome extension that helps you stay on track while cooking.

It reads the ingredients on any recipe page and adds small inline badges directly into the directions. As you cook, you can tap each badge to mark that ingredient as "done", so you always know what you've already used and what comes next.

---

## What RecipeFix does

- **Reads ingredients on recipe pages**  
  When you open a recipe, RecipeFix looks for the ingredient list and the step‑by‑step directions on that page.

- **Adds inline ingredient badges into directions**  
  Wherever an ingredient appears in a step, RecipeFix adds a small badge that matches the ingredient from the list.

- **Lets you tap to mark each one done**  
  As you cook, simply tap a badge to mark it done. This creates a gentle, built‑in checklist right inside the directions, so you don’t have to scroll up and down to double‑check what you’ve already used.

- **Works across many recipe sites**  
  RecipeFix is designed to run on all URLs so it can support recipes from a wide variety of websites you already use.

---

## Who it’s for

RecipeFix is for home cooks of all levels who:

- Get lost jumping between long ingredient lists and multi‑step directions.
- Worry about accidentally skipping an ingredient partway through a recipe.
- Prefer visual checkmarks and gentle reminders over complicated cooking apps.

If you read recipes in your browser and want a calmer, more organized way to cook, RecipeFix is made for you.

---

## How to use it

1. **Install the extension**  
   Add RecipeFix from the Chrome Web Store (search for “RecipeFix”).

2. **Open any online recipe**  
   Visit one of your usual recipe sites in Chrome.

3. **Scroll to the directions**  
   As you read, you’ll see small badges appear next to ingredients mentioned in each step.

4. **Tap badges as you cook**  
   Each time you use an ingredient, tap its badge to mark it done. The badge will update, so you can see at a glance which ingredients you’ve already used.

## Privacy

RecipeFix is designed to be simple and respectful of your privacy:

- **No personal data collection**  
  RecipeFix does not collect, store, or share any personal information about you. There are no accounts, logins, or profiles.

- **No analytics or tracking**  
  RecipeFix does not use analytics tools or track which sites or recipes you visit.

- **No remote code execution**  
  The extension does not download or run code from remote servers. All of its logic is packaged with the extension and runs locally in your browser.

- **Why it runs on all sites**  
  RecipeFix requests permission to run on all websites so it can work on recipes from many different sites. This access is used only to:
  - Detect when you are on a recipe.
  - Read the ingredients and directions on that page.
  - Insert inline badges into the directions while you cook.

  The page content it reads is processed locally in your browser and is not sent anywhere.
